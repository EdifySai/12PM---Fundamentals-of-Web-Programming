HTML Lists:

To list a set of items on the UI, we have two types of lists. 

1. Unordered List :

tag: <ul>

<html>

<head>

</head>

<body>

    <ul style="list-style-type: disc">
        <li>Item 1</li>
        <ul style="list-style-type: square">
            <li>Item 1.1</li>
            <li>Item 1.2</li>
            <li>Item 1.3</li>
            <li>Item 1.4</li>
        </ul>
        <li>Item 2</li>
        <li>Item 3</li>
        <li>Item 4</li>
        <li>Item 5</li>
    </ul>

</body>

</html>


2. Ordered List 

tag: <ol>

<html>

<head>

</head>

<body>
    <ol start="4">
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
        <li>Item 4</li>
        <li>Item 5</li>
    </ol>
</body>

</html>
