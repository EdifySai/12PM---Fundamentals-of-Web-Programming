file paths: 

There are 2 types of file paths.

1. absolute  : Path from the base root (complete path)

/Users/saikumar/Desktop/12pm-web/12PM---Fundamentals-of-Web-Programming/22July2023/images/logos/logo.jpeg

2. relative  : path from the current directiory

12PM---Fundamentals-of-Web-Programming/22July2023/images/logos/logo.jpeg
